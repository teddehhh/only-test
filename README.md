# Historical Dates (Исторические Даты)

Интерактивное веб-приложение для визуализации временных периодов и ключевых исторических событий. Проект выполнен на **React** с использованием современных библиотек для анимации и слайдинга.

## Архитектура приложения

Проект реализован в соответствии с методологией Feature-Sliced Design (FSD), что обеспечивает слабую связанность модулей и высокую масштабируемость.

Описание слоев, слайсов и сегментов ниже в основном **опирается на директории и файлы в текущем проекте**. Подробнее о том, что может храниться в разных слоях, слайсах и сегментах можно почитать [здесь](https://feature-sliced.design/ru/docs/get-started/overview). 

## Слои:
### app

Отвечает за инициализацию приложения. Содержит глобальные стили `styles`, входную точку `main.tsx` и корневой компонент `app.tsx`.

### pages
Содержит единственную и главную страницу проекта `main`. Она включает виджет для формирования главной страницы проекта.

### widgets
Хранит `history-block` — ключевой слайс приложения. Содержит композицию из вложенных компонентов и логики управления периодами.

`ui` — визуальные компоненты виджета (кольцо периодов, блок навигации).

`lib` — вспомогательные функции, хуки и данные, специфичные для этого виджета.

### entities
Включает в себя бизнес-сущность `history-event` из слайдера.

`ui` — содержит визуальный компонент.

`model` — содержит типы данных.


### shared
`shared` — хранит переиспользуемый код.

`ui` — базовые компоненты (кнопки, иконки).

`lib` — утилиты общего назначения (например, проверка текущей ширины экрана).

`assets` — статические ресурсы (шрифты, изображения).

`config` — конфигурационные файлы проекта.

## Быстрый старт

### Предварительные требования

Необходимая версия Node.js 24.12.0

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/teddehhh/only-test
   ```
2. Перейдите в директорию проекта
   ```bash
   cd only-test
   ```
3. Установите зависимости
   ```bash
   npm i
    # или
   yarn i
   ```

### Запуск

1. Запустите сервер
   ```bash
   npm run dev
   # или
   yarn run dev
   ```
2. Откройте http://localhost:8080 в браузере.
